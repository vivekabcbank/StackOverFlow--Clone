How to use the Flag Post queue
Awarded at 2k reputation

Evaluate proposed edits by users without enough reputation to make direct changes to other users' posts.

You can Approve, Improve, or Reject edits.

How to work through this queue:

  • 2 users must approve an edit for the post to publish.
  • Edits should maintain the post author’s original intent.
  • Reject edits that are spam, attempt to reply to the post author, or clearly worsen the post.
  • Even small changes can be good edits! Choose Improve edit if a post could use more changes.
  • Skip the task if you aren't sure which action to take.
Read more about this queue in our help center.
